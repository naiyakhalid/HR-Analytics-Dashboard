# HR-Analytics-Dashboard

## Project objective: 
The objective of this HR Interactive Dashboard is to provide a comprehensive overview of employee data to support data-driven decision-making in Human Resources. The dashboard visualizes key workforce metrics such as salary distribution, gender diversity, department headcount, performance-linked compensation, age distribution, and tenure patterns.It enables stakeholders to:

‚Ä¢ Identify trends in compensation and workforce demographics
‚Ä¢ Assess pay equity across performance and job levels
‚Ä¢ Monitor workforce composition by gender, age, and department

# Project Insights
‚Ä¢ Salary Concentration: Most employees earn between $40,000 and $100,000, as shown by the salary distribution histogram.
‚Ä¢ Gender Composition: The workforce consists of 55% Male and 45% Female, indicating near balance but worth monitoring for equity at senior levels.
